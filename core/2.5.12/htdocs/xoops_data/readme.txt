Para los datos de XOOPS

1)  Por motivos de seguridad se recomienda encarecidamente mover esta carpeta fuera del Document Root.

2) Conceda permisos de escritura a esta carpeta.

3) Asegúrese de que las siguientes subcarpetas existan y sean escribibles:
        configs/
        configs/captcha/
        configs/textsanizer/
        caches/
        caches/xoops_cache/
        caches/smarty_cache/
        caches/smarty_compile/
        data/
        protector/

Nota: Estos directorios ya deben existir y se copian desde el archivo de instalación. Esa copia incluye archivos críticos.

Para la documentación completa sobre instalación o actualización de XOOPS visite:
https://xoops.gitbook.io/xoops-install-upgrade/
