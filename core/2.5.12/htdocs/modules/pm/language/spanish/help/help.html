Ayuda: Mensajería Privada

DESCRIPCIÓN

El módulo PM es para la mensajería privada entre usuarios

INSTALAR/DESINSTALAR

No son necesarias medidas especiales, siga el proceso de instalación estándar: extraiga la carpeta /pm en el directorio ../modules. Instale el módulo a través de Administración -> Módulo del Sistema -> Módulos.

Las instrucciones detalladas sobre la instalación de módulos están disponibles en el Manual de Operaciones de XOOPS

INSTRUCCIONES DE FUNCIONAMIENTO

Para configurar este módulo, necesita:

  • Ingrese a su Bandeja de entrada directamente desde la página principal, o haciendo clic en el enlace superior "Ir al módulo" en el área de Administración de PM.
  • Configure sus preferencias para el módulo (consulte "Preferencias") y, opcionalmente, el bloque PM si tiene la intención de usarlo (consulte "Bloques").
  • Puede eliminar masivamente sus mensajes utilizando la pestaña "Eliminar Mensajes".
