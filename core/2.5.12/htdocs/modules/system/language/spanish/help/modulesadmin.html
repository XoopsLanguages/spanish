Ayuda: Módulos

Descripción

Los módulos son las construcciones de software que, aprovechando la infraestructura que proporciona XOOPS, permiten al webmaster ofrecer a los usuarios del sitio contenido dinámico o estático generado mediante una interfaz web que, la mayoría de las veces, es muy fácil de usar.

Parte del poder de XOOPS reside en la forma en que se gestionan los módulos. El webmaster puede instalar, desinstalar, activar o desactivar fácilmente cualquier módulo específico (excepto el módulo del Sistema, por razones obvias). También es posible cambiar el nombre de los módulos y determinar el orden en que aparecerán al usuario final en el menú del sitio (si es que desea que aparezcan), porque también puede proporcionar funcionalidad oculta, definiendo módulos activos que no tienen presencia visible para el usuario ocasional. Por ejemplo, podría definir un módulo como no visible en el menú desactivando el icono de verificación en la columna Menú. Entonces, para el módulo de Noticias, podría enviar mensajes privados a un grupo seleccionado de sus usuarios y decirles que accedan a ese directorio escribiendo la dirección en el campo URL, algo así como http://www.susitio.com/modules/news.

Para cambiar el orden de los módulos (que se reflejará en el Menú), solo necesita arrastrar y soltar los módulos en la ubicación deseada.

La instalación del módulo necesita cierta planificación, tanto para los módulos que se mostrarán como para el diseño de las páginas. Es una buena idea revisar otros sitios de XOOPS y ver qué módulos se utilizan para mostrar varios tipos de contenido. Esto se hace fácilmente abriendo cualquier sitio de XOOPS y pasando el cursor sobre los enlaces del menú en los paneles laterales izquierdo y derecho. Al mismo tiempo, observe las URL que se muestran en la barra de estado de su navegador. Debería ver direcciones como esta: http://el_sitio.com/modules/news/.... Todos los módulos se almacenan en el directorio /modules/, por lo que el siguiente sector de la dirección (en este ejemplo 'news') muestra el módulo que mostrará el contenido de ese enlace del menú.

Si abre la página del enlace, mostrará un ejemplo de cómo se utiliza el módulo y el tipo de contenido que muestra. Revisar otros sitios como este también le dará una buena idea de las capacidades de diseño y maquetación.
