Ayuda: Mantenimiento

Descripción

El mantenimiento proporciona varias funciones básicas para ayudarte a mantener tu instalación de XOOPS y mantenerla en buen estado:

- Limpiar la carpeta de caché
- Vaciar la tabla de sesiones
- Realizar el mantenimiento de las tablas de la base de datos (analizar, verificar, optimizar y reparar)

También puedes realizar un "Volcado" de toda la base de datos o de tablas individuales con fines de respaldo.
