Bienvenido al: Centro de Ayuda de XOOPS

¿Cómo uso XOOPS?

XOOPS es una plataforma de aplicación web escrita en PHP para la base de datos MySQL. Su orientación a objetos la convierte en una herramienta ideal para desarrollar sitios web comunitarios pequeños o grandes, portales intraempresariales y corporativos, weblogs y mucho más. Para usar XOOPS de manera efectiva, necesita instalar módulos que proporcionen la funcionalidad que está buscando. Hay muchos módulos diferentes en Repositorio de Módulos, desde calendarios, galerías de fotos, pasando por noticias, tablones de anuncios, hasta actualizaciones meteorológicas.

¿Cómo uso el Centro de Ayuda?

El Centro de Ayuda está para proporcionarle información básica relacionada con las características individuales de la Administración. Siempre que desee saber más sobre una página en particular en la que se encuentre, simplemente haga clic en el icono de ayuda en la esquina derecha

¿Cómo puedo crear contenido en XOOPS?

Necesitará un módulo para ello. Esto podría ser un módulo de Noticias o un módulo de Contenido.
