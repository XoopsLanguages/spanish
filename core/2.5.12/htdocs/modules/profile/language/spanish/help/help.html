Ayuda: Perfil

DESCRIPCIÓN

El módulo de Perfil es para la gestión de campos personalizados del perfil de usuario.

INSTALAR/DESINSTALAR

No son necesarias medidas especiales, siga el proceso de instalación estándar: extraiga la carpeta /profile en el directorio ../modules. Instale el módulo a través de Administración -> Módulo del Sistema -> Módulos.

Las instrucciones detalladas sobre la instalación de módulos están disponibles en el Manual de Operaciones de XOOPS

INSTRUCCIONES DE FUNCIONAMIENTO

Para configurar este módulo, necesita:

  • Configure sus preferencias para el módulo (consulte 'Preferencias') y, opcionalmente, el bloque de Perfil si tiene la intención de usarlo (consulte 'Bloques').
  • Edite las Categorías existentes o agregue nuevas.
  • Edite los Campos existentes o agregue nuevos. Aquí puede especificar qué campos serán visibles en qué categoría y si serán visibles durante el registro de usuarios.
  • Defina el orden de los pasos de Registro.
  • Y finalmente, puede establecer permisos para campos individuales: cuáles son editables, cuáles se pueden buscar.

Medidas Anti-Spam

Para minimizar los registros de spam, haga lo siguiente:

  • vaya al módulo Protector en Administración, vaya a Preferencias y luego, en la parte inferior, en esta opción: "Detener el Spam del Foro" Verifica los datos POST contra los spammers registrados en la base de datos www.stopforumspam.com.
    Establézcalo en "Prohibir la IP (sin límite)"

  • en /class/captcha/config.php, asegúrese de que el modo esté establecido como "text":

    return $config = array(
    'disabled' => false, // Desactivar CAPTCHA
    'mode' => 'text', // modo predeterminado, puede elegir 'text', 'image', 'recaptcha' (requiere clave api)
    'name' => 'xoopscaptcha', // nombre de captcha
    'skipmember' => true, // Omitir la verificación CAPTCHA para los miembros
    'maxattempts' => 10, // Máximo de intentos por sesión
    );

  • En el módulo de Perfil, vaya a Administración y en el paso Básico ubicado en:
    /modules/profile/admin/step.php?id=1
    establezca "Guardar después de detener" en "No"


  • En Preferencias de Perfil, establezca la Opción "Usar Captcha después del segundo paso de Registro" en "Sí" (es el valor predeterminado)
