Instalación de XOOPS

Instalación por primera vez

Prefacio:

XOOPS es un sistema de publicación web de código abierto orientado a objetos, escrito en PHP. Es una herramienta ideal para desarrollar sitios web comunitarios dinámicos de pequeño a gran tamaño, portales internos de empresas, portales corporativos, weblogs y mucho más.

XOOPS se distribuye bajo los términos de la Licencia Pública General de GNU (GPL) versión 2 o superior, y es gratuito para usar y modificar. Puede redistribuirse libremente siempre que se cumplan los términos de distribución de la GPL.

Requisitos

  • Servidor WWW (Apache, NGINX, IIS, etc.)
  • PHP 7.4 o superior, se recomienda 7.4 o superior
  • MySQL 5.7 o superior, se recomienda MySQL 8.0 o superior

Antes de instalar

  1. Configure correctamente el servidor WWW, PHP y el servidor de base de datos.
  2. Prepare una base de datos para su sitio XOOPS.
  3. Prepare una cuenta de usuario y otorgue a ese usuario acceso a la base de datos.
  4. Haga que los directorios y archivos sean escribibles: %s
  5. Por motivos de seguridad, se recomienda encarecidamente mover los dos directorios mencionados a continuación fuera del document root y cambiar los nombres de las carpetas: %s
  6. Cree (si aún no existen) y haga escribibles los directorios: %s
  7. Active las cookies y JavaScript en su navegador.

Instalación local

Si está ejecutando un entorno local para desarrollo o pruebas, asegúrese de que se cumplan los requisitos anteriores. Una vez hecho esto, copie el contenido del directorio htdocs (del archivo de distribución de XOOPS) al directorio raíz de documentos de su entorno web. Una vez que los archivos estén copiados allí, puede iniciar la instalación dirigiendo su navegador a "http://su-sitio.com". Esto iniciará el proceso de instalación.

Instalación en una plataforma alojada

Si está trabajando en un entorno alojado, descomprima los archivos de XOOPS localmente o en el servidor si tiene acceso a shell (por ejemplo, SSH). Una vez hecho esto, asegúrese de mover o copiar todos los archivos de XOOPS desde el directorio htdocs a su directorio web raíz (su proveedor generalmente proporciona esta ubicación con instrucciones). Una vez que los archivos estén copiados allí, puede iniciar la instalación dirigiendo su navegador a "http://su-sitio.com". Esto iniciará el proceso de instalación.

Continuar con la instalación

Solo siga las instrucciones paso a paso del instalador.
